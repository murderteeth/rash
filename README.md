# rash
A CLI tool that translates natural language into bash commands.

## setup
```bash
cp .env.example .env
# configure .env
yarn
yarn build
```

## test
```bash
yarn console
```

## install
```bash
npm install -g .
# or
yarn global add file:$PWD
```

## uninstall
```bash
npm uninstall -g rash
# or
yarn global remove rash
```
